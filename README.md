# 🔐 Password Generator
A customizable password generator built with vanilla HTML, CSS, and JavaScript.

🔗 [Live Demo](https://adamkirikisla.github.io/password-generator/)

## Features

- Adjustable password length (1–32 characters)
- Toggle character types: uppercase, lowercase, numbers, symbols
- Password strength indicator
- One-click copy to clipboard

## Built With

- HTML
- CSS (glassmorphism-style UI)
- JavaScript (DOM manipulation, no frameworks/libraries)
